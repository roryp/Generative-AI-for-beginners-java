Azure AI Foundry provides a unified platform to access and use Large Language Models (LLMs) with keyless authentication through Microsoft Entra ID.
This service supports various AI capabilities including chat completions and text embeddings, making it accessible for developers to integrate AI features into their applications.
The platform runs on Microsoft Azure infrastructure, ensuring reliability, security, and scalability. With Azure AI Foundry, developers can quickly prototype and build AI-powered applications by provisioning model deployments as code with Bicep and the Azure Developer CLI (azd), then authenticating with their Azure credentials using the Cognitive Services OpenAI User role.
