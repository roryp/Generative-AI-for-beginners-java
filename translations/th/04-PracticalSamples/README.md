## สิ่งที่คุณจะได้เรียนรู้
ในส่วนนี้เราจะแสดงตัวอย่างสามแอปพลิเคชันเชิงปฏิบัติที่แสดงรูปแบบการพัฒนา AI สร้างสรรค์ด้วย Java:
- สร้างเครื่องสร้างเรื่องราวสัตว์เลี้ยงแบบหลายโหมดผสมผสาน AI ฝั่งไคลเอ็นต์และฝั่งเซิร์ฟเวอร์
- นำเสนอบูรณาการโมเดล AI ท้องถิ่นด้วย Foundry Local Spring Boot demo
- พัฒนาบริการโปรโตคอลบริบทโมเดล (MCP) ด้วยตัวอย่างเครื่องคิดเลข

## บทนำ

บทนี้แสดง **โครงการตัวอย่าง** ที่แสดงรูปแบบการพัฒนา AI สร้างสรรค์ด้วย Java แต่ละโครงการทำงานได้สมบูรณ์และแสดงเทคโนโลยี AI เฉพาะ รูปแบบสถาปัตยกรรม และแนวทางปฏิบัติที่ดีที่สุดที่คุณสามารถปรับใช้กับแอปของคุณเองได้

### Foundry Local Spring Boot Demo

**[Foundry Local Spring Boot Demo](foundrylocal/README.md)** แสดงวิธีเชื่อมต่อกับโมเดล AI ท้องถิ่นโดยใช้ **OpenAI Java SDK** ซึ่งแสดงการเชื่อมต่อกับโมเดลที่รันบน Foundry Local (เช่น **Phi-4-mini**) พร้อมการตรวจจับโมเดลอัตโนมัติ ช่วยให้คุณรันแอป AI ได้โดยไม่ต้องพึ่งพาบริการคลาวด์

### เครื่องสร้างเรื่องราวสัตว์เลี้ยง

**[เครื่องสร้างเรื่องราวสัตว์เลี้ยง](petstory/README.md)** คือแอปพลิเคชันเว็บ Spring Boot ที่น่าสนใจซึ่งแสดงการประมวลผล AI แบบหลายโหมดเพื่อสร้างเรื่องราวสัตว์เลี้ยงที่สร้างสรรค์ โดยผสมผสานความสามารถ AI ฝั่งไคลเอ็นต์และฝั่งเซิร์ฟเวอร์โดยใช้ transformer.js สำหรับการโต้ตอบ AI บนเบราว์เซอร์ และ OpenAI SDK สำหรับประมวลผลฝั่งเซิร์ฟเวอร์

### บริการ MCP Calculator (ตัวอย่าง MCP สำหรับผู้เริ่มต้น)

**[บริการ MCP Calculator](calculator/README.md)** เป็นการสาธิตง่ายๆ ของ **โปรโตคอลบริบทโมเดล (MCP)** โดยใช้ Spring AI ซึ่งนำเสนอแนวคิดพื้นฐานของ MCP สำหรับผู้เริ่มต้น แสดงวิธีสร้าง MCP Server เบื้องต้นที่โต้ตอบกับ MCP clients

## ลำดับการเรียนรู้

โครงการเหล่านี้ออกแบบมาเพื่อสร้างความเข้าใจจากบทก่อนหน้า:

1. **เริ่มต้นง่ายๆ**: เริ่มด้วย Foundry Local Spring Boot Demo เพื่อทำความเข้าใจการบูรณาการ AI กับโมเดลท้องถิ่นขั้นพื้นฐาน
2. **เพิ่มการโต้ตอบ**: ก้าวไปยังเครื่องสร้างเรื่องราวสัตว์เลี้ยงสำหรับ AI แบบหลายโหมดและการโต้ตอบบนเว็บ
3. **เรียนรู้พื้นฐาน MCP**: ทดลองบริการ MCP Calculator เพื่อเข้าใจพื้นฐานโปรโตคอลบริบทโมเดล

## สรุป

ทำได้ดีมาก! ตอนนี้คุณได้สำรวจแอปจริงบางส่วนแล้ว:

- ประสบการณ์ AI แบบหลายโหมดที่ทำงานได้ทั้งบนเบราว์เซอร์และบนเซิร์ฟเวอร์
- การบูรณาการโมเดล AI ท้องถิ่นโดยใช้เฟรมเวิร์กและ SDK Java ที่ทันสมัย
- บริการโปรโตคอลบริบทโมเดลแรกของคุณเพื่อดูวิธีเครื่องมือทำงานร่วมกับ AI
