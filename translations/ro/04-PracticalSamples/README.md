## Introducere

Acest capitol prezintă **proiecte exemplu** care demonstrează tipare de dezvoltare AI generativă cu Java. Fiecare proiect este complet funcțional și demonstrează tehnologii AI specifice, tipare arhitecturale și bune practici pe care le poți adapta pentru propriile aplicații.

### Demo Foundry Local Spring Boot

**[Demo Foundry Local Spring Boot](foundrylocal/README.md)** demonstrează cum să integrezi modele AI locale folosind **OpenAI Java SDK**. Prezintă conectarea la modele care rulează pe Foundry Local (de exemplu, **Phi-4-mini**), cu detectare automată a modelului, permițând rularea aplicațiilor AI fără a depinde de servicii cloud.

### Generator de povestiri pentru animale de companie

**[Generator de povestiri pentru animale de companie](petstory/README.md)** este o aplicație web Spring Boot captivantă care demonstrează **procesarea AI multi-modală** pentru a genera povești creative despre animale de companie. Combină capacități AI client-side și server-side folosind transformer.js pentru interacțiuni AI în browser și OpenAI SDK pentru procesare pe server.

### Serviciu MCP Calculator (demo MCP prietenos pentru începători)

**[Serviciul MCP Calculator](calculator/README.md)** este o demonstrație simplă a **Model Context Protocol (MCP)** folosind Spring AI. Oferă o introducere prietenoasă pentru începători în conceptele MCP, arătând cum să creezi un server MCP de bază care interacționează cu clienți MCP.

## Progresul în învățare

Aceste proiecte sunt concepute pentru a se construi pe conceptele din capitolele anterioare:

1. **Începe simplu**: Începe cu demo-ul Foundry Local Spring Boot pentru a înțelege integrarea de bază AI cu modele locale
2. **Adaugă interactivitate**: Progresează la Generatorul de povestiri pentru a învăța AI multi-modal și interacțiuni web
3. **Învață noțiunile MCP**: Încearcă Serviciul MCP Calculator pentru a înțelege fundamentele Model Context Protocol

## Sumar

Bravo! Acum ai explorat unele aplicații reale:

- Experiențe AI multi-modal care funcționează atât în browser, cât și pe server
- Integrare locală a modelelor AI folosind cadre moderne Java și SDK-uri
- Primul tău serviciu Model Context Protocol pentru a vedea cum instrumentele interacționează cu AI
