## Introdução

Este capítulo apresenta **projetos exemplares** que demonstram padrões de desenvolvimento AI generativa com Java. Cada projeto é totalmente funcional e demonstra tecnologias específicas de AI, padrões arquiteturais e boas práticas que pode adaptar às suas próprias aplicações.

### Demo Foundry Local Spring Boot

A **[Demo Foundry Local Spring Boot](foundrylocal/README.md)** demonstra como integrar com modelos AI locais usando o **OpenAI Java SDK**. Mostra a ligação a modelos a correr no Foundry Local (ex., **Phi-4-mini**), com deteção automática de modelos, permitindo executar aplicações AI sem depender de serviços na cloud.

### Gerador de Histórias de Animais

O **[Gerador de Histórias de Animais](petstory/README.md)** é uma aplicação web Spring Boot envolvente que demonstra **processamento AI multi-modal** para gerar histórias criativas sobre animais. Combina capacidades AI do lado do cliente e do servidor usando transformer.js para interações AI no browser e o SDK OpenAI para processamento no servidor.

### Serviço MCP Calculadora (Demo MCP acessível para iniciantes)

O **[Serviço MCP Calculadora](calculator/README.md)** é uma demonstração simples do **Model Context Protocol (MCP)** usando Spring AI. Proporciona uma introdução acessível aos conceitos MCP, mostrando como criar um MCP Server básico que interage com clientes MCP.

## Progressão da Aprendizagem

Estes projetos foram desenhados para construir conhecimentos a partir dos capítulos anteriores:

1. **Começar Simples**: Comece com a Demo Foundry Local Spring Boot para entender a integração básica AI com modelos locais
2. **Adicionar Interatividade**: Progrida para o Gerador de Histórias de Animais para AI multi-modal e interações baseadas na web
3. **Aprender noções básicas de MCP**: Experimente o Serviço MCP Calculadora para compreender os fundamentos do Model Context Protocol
