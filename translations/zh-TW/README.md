**時間投入**：整個工作坊可在線完成，無需本地環境搭建。環境設置約需 2 分鐘，探索範例則依探索深度需 1-3 小時。

> **快速開始**

1. 將此儲存庫 Fork 到你的 GitHub 帳號
2. 點擊 **Code** → **Codespaces** 分頁 → **...** → **New with options...**
3. 使用預設設定 – 會自動選擇本課程建立的開發容器
4. 點擊 **Create codespace**
5. 等待約 2 分鐘，環境即準備完成
6. 直接跳到 [第一個範例](./02-SetupDevEnvironment/README.md#step-2-create-a-github-personal-access-token)

> **想要本地 Clone？**
>
> 此儲存庫包含 50 多種語言翻譯，顯著增加下載大小。若想不含翻譯檔案下載，可使用稀疏檢出方式：
>
> **Linux / macOS (Bash)**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/Generative-AI-for-beginners-java.git
> cd Generative-AI-for-beginners-java
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **Windows (PowerShell)**
> ```powershell
> git clone --filter=blob:none --sparse https://github.com/microsoft/Generative-AI-for-beginners-java.git
> cd Generative-AI-for-beginners-java
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
> 這樣你可以更快下載到完成課程所需的全部內容。

## 課程架構與學習路徑

### **第一章：生成式 AI 介紹**
- **核心概念**：了解大型語言模型、詞元、嵌入向量與 AI 能力
- **Java AI 生態系**：Spring AI 和 OpenAI SDK 簡介
- **模型上下文協議**：MCP 及其在 AI 代理間通訊的角色介紹
- **實務應用**：包含聊天機器人及內容生成的實際案例
- **[→ 開始第一章](./01-IntroToGenAI/README.md)**

### **第二章：開發環境設定**
- **多供應商配置**：GitHub Models、Azure OpenAI 與 OpenAI Java SDK 整合設定
- **Spring Boot + Spring AI**：企業級 AI 應用開發最佳實踐
- **GitHub Models**：免費 AI 模型供原型與學習使用（無需信用卡）
- **開發工具**：Docker 容器、VS Code 與 GitHub Codespaces 配置
- **[→ 開始第二章](./02-SetupDevEnvironment/README.md)**

### **第三章：生成式 AI 核心技術**
- **提示詞工程**：取得最佳 AI 模型回應的技巧
- **嵌入向量與向量運算**：實作語意搜尋與相似度比對
- **檢索擴增生成 (RAG)**：結合 AI 與自有資料源
- **函式呼叫**：透過自訂工具與插件擴展 AI 功能
- **[→ 開始第三章](./03-CoreGenerativeAITechniques/README.md)**

### **第四章：實務應用與專案**
- **寵物故事生成器** (`petstory/`)：利用 GitHub Models 的創意內容生成
- **Foundry 本地示範** (`foundrylocal/`)：OpenAI Java SDK 本地 AI 模型整合
- **MCP 計算服務** (`calculator/`)：使用 Spring AI 實作基礎模型上下文協定
- **[→ 開始第四章](./04-PracticalSamples/README.md)**

### **第五章：負責任的 AI 開發**
- **GitHub Models 安全性**：測試內建內容過濾與安全機制（硬封鎖與軟拒絕）
- **負責任 AI 示範**：實作演練現代 AI 安全系統如何運作
- **最佳實務**：倫理 AI 開發與部署的必要指導方針
- **[→ 開始第五章](./05-ResponsibleGenAI/README.md)**
