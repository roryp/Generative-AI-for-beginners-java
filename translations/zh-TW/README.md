**時間投入**：整個工作坊可以在線完成，無需本地設置。環境設置約需 2 分鐘，探索範例依深度需 1-3 小時。

> **快速開始**

1. 將此儲存庫 Fork 至您的 GitHub 帳號
2. 點擊 **Code** → **Codespaces** 分頁 → **...** → **New with options...**
3. 使用預設值 — 這會選擇為本課程建立的開發容器
4. 點擊 **Create codespace**
5. 等待約 2 分鐘，環境即準備就緒
6. 直接跳至 [第一個範例](./02-SetupDevEnvironment/README.md#step-2-create-a-github-personal-access-token)

> **偏好本地克隆？**
>
> 此儲存庫包含 50 多種語言翻譯，大大增加下載大小。若想不下載翻譯，請使用稀疏簽出：
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/Generative-AI-for-beginners-java.git
> cd Generative-AI-for-beginners-java
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
> 這將給您完成課程所需的一切，且下載速度更快。

## 課程架構與學習路徑

### **第一章：生成式 AI 簡介**
- **核心概念**：理解大型語言模型、標記、嵌入向量與 AI 能力
- **Java AI 生態系統**：Spring AI 與 OpenAI SDK 概覽
- **模型上下文協定（MCP）**：介紹 MCP 與 AI 代理通訊的角色
- **實務應用**：真實案例，包括聊天機器人與內容生成
- **[→ 開始第一章](./01-IntroToGenAI/README.md)**

### **第二章：開發環境設置**
- **多供應商配置**：設置 GitHub Models、Azure OpenAI 與 OpenAI Java SDK 整合
- **Spring Boot + Spring AI**：企業 AI 應用最佳實踐
- **GitHub Models**：免費 AI 模型，用於原型設計和學習（免信用卡）
- **開發工具**：Docker 容器、VS Code 與 GitHub Codespaces 設定
- **[→ 開始第二章](./02-SetupDevEnvironment/README.md)**

### **第三章：核心生成式 AI 技術**
- **提示工程**：提升 AI 模型回應品質的技巧
- **嵌入向量與向量運算**：實作語意搜尋與相似度匹配
- **檢索增強生成（RAG）**：將 AI 與自有資料來源結合
- **函式呼叫**：以自訂工具與插件擴展 AI 功能
- **[→ 開始第三章](./03-CoreGenerativeAITechniques/README.md)**

### **第四章：實務應用與專案**
- **寵物故事產生器** (`petstory/`)：用 GitHub Models 創造內容
- **Foundry Local Demo** (`foundrylocal/`)：本地 AI 模型整合 OpenAI Java SDK
- **MCP 計算機服務** (`calculator/`)：基礎模型上下文協定實作及 Spring AI
- **[→ 開始第四章](./04-PracticalSamples/README.md)**

### **第五章：負責任 AI 開發**
- **GitHub Models 安全性**：測試內建內容過濾與安全機制（嚴格阻擋及軟性拒絕）
- **負責任 AI 示範**：操作範例呈現現代 AI 安全系統實務運作
- **最佳實務**：倫理 AI 開發與部署的重要指導方針
- **[→ 開始第五章](./05-ResponsibleGenAI/README.md)**
