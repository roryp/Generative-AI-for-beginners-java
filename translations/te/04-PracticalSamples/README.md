## మీరు నేర్చుకునేది ఏమిటి
ఈ విభాగంలో మేము జావాతో జనరేటివ్ AI అభివృద్ధి నమూనాలను ప్రదర్శించే మూడు ప్రాక్టికల్ అప్లికేషన్లను డెమో చేస్తాము:
- క్లయింట్-సైడ్ మరియు సర్వర్-సైడ్ AI ను కలిపిన మల్టీ-మోడ్ పెట్ స్టోరీ జనరేటర్ సృష్టించండి
- Foundry Local Spring Boot డెమోతో లోకల్ AI మోడల్ ఇంటిగ్రేషన్ అమలు చేయండి
- కాలిక్యులేటర్ ఉదాహరణతో మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP) సేవను అభివృద్ధి చేయండి

## పరిచయం

ఈ అధ్యాయం **జావాతో జనరేటివ్ AI అభివృద్ధి నమూనాలను ప్రదర్శించే నమూనా ప్రాజెక్ట్ లను** చూపిస్తుంది. ప్రతి ప్రాజెక్ట్ పూర్తిగా నిర్వహించదగినది మరియు మీరు మీ స్వంత అప్లికేషన్ల కోసం అనుకరించుకునే ప్రత్యేక AI సాంకేతికతలు, వాస్తవ్య నమూనాలు మరియు ఉత్తమ ఆచరణలను ప్రదర్శిస్తుంది.

### Foundry Local Spring Boot డెమో

**[Foundry Local Spring Boot డెమో](foundrylocal/README.md)** లో **OpenAI Java SDK** ఉపయోగించి లోకల్ AI మోడల్స్ తో ఎలా ఇంటిగ్రేట్ చేయాలో చూపిస్తుంది. ఇది Foundry Local (ఉదా., **Phi-4-mini**)పై నడుస్తున్న మోడల్స్ కు కనెక్ట్ అయి ఆటోమేటిక్ మోడల్ డిటెక్షన్ ను ప్రదర్శిస్తుంది, క్లౌడ్ సేవల మీద ఆధారపడకుండా AI అప్లికేషన్లను నడపడానికి అనుమతిస్తుంది.

### పెట్ స్టోరీ జనరేటర్

**[పెట్ స్టోరీ జనరేటర్](petstory/README.md)** అనేది ఒక ఆకట్టుకునే Spring Boot వెబ్ అప్లికేషన్, ఇది **మల్టీ-మోడ్ AI ప్రాసెసింగ్** ను ఉపయోగించి సృజనాత్మక పెట్ స్టొరీలను తయారు చేస్తుంది. ఇది క్లయింట్-సైడ్ మరియు సర్వర్-సైడ్ AI సామర్ధ్యాలను కలిపి, బ్రౌజర్ ఆధారిత AI ఇంటరాక్షన్ల కోసం transformer.js మరియు సర్వర్-సైడ్ ప్రాసెసింగ్ కోసం OpenAI SDKని ఉపయోగిస్తుంది.

### MCP కాలిక్యులేటర్ సేవ (ప్రారంభిక-స్నేహపూర్వక MCP డెమో)

**[MCP కాలిక్యులేటర్ సేవ](calculator/README.md)** అనేది Spring AI ఉపయోగించి **మోడల్ కాంటెక్స్ట్ ప్రోటోకాల్ (MCP)** సందర్శన. ఇది MCP సూత్రాలపై ప్రారంభికులకు సురక్షిత పరిచయాన్ని అందిస్తుంది, MCP క్లయింట్లతో పరస్పరం పనిచేసే ఒక ప్రాథమిక MCP సర్వర్ సృష్టించడం ఎలా అనేది చూపిస్తుంది.

## అర్ధం చేసుకునే ప్రగతి

ఈ ప్రాజెక్టులు మునుపటి అధ్యాయాల నుండి సూత్రాలపై నిర్మించబడేలా రూపొందించబడ్డాయి:

1. **సులభంగా ప్రారంభించండి:** Foundry Local Spring Boot డెమోతో లోకల్ మోడల్స్‌తో AI ఇంటిగ్రేషన్ యొక్క ప్రాథమికాలను అర్థం చేసుకోండి
2. **ఇంటరాక్టివిటీని జోడించండి:** మల్టీ-మోడ్ AI మరియు వెబ్-ఆధారిత ఇంటరాక్షన్ల కొరకు పెట్ స్టోరీ జనరేటర్ కు ముందుకు పోయి
3. **MCP ప్రాథమికాలు నేర్చుకోండి:** Model Context Protocol మౌలیک అంశాలను అర్థం చేసుకోవడానికి MCP కాలిక్యులేటర్ సేవను ప్రయత్నించండి

## సారాంశం

చాలా బాగుంది! మీరు ఇప్పుడు నిజమైన కొన్ని అప్లికేషన్లను అన్వేషించారు:

- బ్రౌజర్ లో మరియు సర్వర్ పై పనిచేసే మల్టీ-మోడ్ AI అనుభవాలు
- ఆధునిక Java ఫ్రేమ్‌వర్క్‌లు మరియు SDKల ఉపయోగంతో లోకల్ AI మోడల్ ఇంటిగ్రేషన్
- ఎలా టూల్స్ AIతో ఇంటిగ్రేట్ అవుతాయో చూడటానికి మీ మొదటి Model Context Protocol సేవ
