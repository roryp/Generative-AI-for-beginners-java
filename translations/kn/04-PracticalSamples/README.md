## ನೀವು ಕಲಿಯಬೇಕಾದವು
ಈ ವಿಭಾಗದಲ್ಲಿ ನಾವು ಮೂರು ಪ್ರಾಯೋಗಿಕ ಅನ್ವಯಿಕೆಗಳನ್ನು ಪ್ರದರ್ಶಿಸುವೆವು, ಅವು ಜಾವಾ ಉಪಯೋಗಿಸಿಕೊ೦ಡಿ ಜನರೇಟಿವ್ AI ಅಭಿವೃದ್ಧಿ ಮಾದರಿಗಳನ್ನು ತೋರಿಸುತ್ತವೆ:
- ಕ್ಲೈಂಟ್-ಸೈಡ್ ಮತ್ತು ಸರ್ವರ್-ಸೈಡ್ AI ಅನ್ನು ಸಂಯೋಜಿಸುವ ಬಹು-ಮೋಡಲ್ ಪೆಟ್ ಸ್ಟೋರಿ ಜೆನರೆಟರ್ ರಚನೆ ಮಾಡಿ
- Foundry Local Spring Boot ಡೆಮೋ ಜೊತೆ ಸ್ಥಳೀಯ AI ಮಾದರಿ ಇಂಟಿಗ್ರೇಶನ್ ಕಾರ್ಯಗತಗೊಳಿಸಿ
- ಕ್ಯಾಲಕರುಲೇಟರ್ ಉದಾಹರಣೆಯೊಂದಿಗೆ ಮಾದರಿ ಸಂತಕ ಮೇಳ (MCP) ಸೇವೆಯನ್ನು ಅಭಿವೃದ್ಧಿಪಡಿಸಿ

## ಪರಿಚಯ

ಈ ಅಧ್ಯಾಯವು **ಉದಾಹರಣಾ ಪ್ರಾಜೆಕ್ಟ್‌ಗಳನ್ನು** ತೋರಿಸುತ್ತದೆ, ಅವು ಜಾವಾ ಉಪಯೋಗಿಸಿದ್ದು ಜನರೇಟಿವ್ AI ಅಭಿವೃದ್ಧಿ ಮಾದರಿಗಳನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತವೆ. ಪ್ರತಿ ಪ್ರಾಜೆಕ್ಟ್ ಸಂಪೂರ್ಣ ಕಾರ್ಯನಿರ್ವಹಣೆಯಾಗಿದೆ ಮತ್ತು ನಿಮಗೆ ಸ್ವಂತ ಅನ್ವಯಿಕೆಗಳಿಗಾಗಿ ಹೊಂದಿಸಬಹುದಾದ ವಿಶಿಷ್ಟ AI ತಂತ್ರಜ್ಞಾನಗಳು, ವಾಸ್ತುಶಿಲ್ಪ ಮಾದರಿಗಳು ಮತ್ತು ಉತ್ತಮ ಅಭ್ಯಾಸಗಳನ್ನು ತೋರಿಸುತ್ತದೆ.

### Foundry Local Spring Boot ಡೆಮೋ

**[Foundry Local Spring Boot ಡೆಮೋ](foundrylocal/README.md)** ಸ್ಥಳೀಯ AI ಮಾದರಿಗಳೊಂದಿಗೆ **OpenAI Java SDK** ಉಪಯೋಗಿಸಿ ಸಂಬಂಧಿಸುವುದನ್ನು ತೋರಿಸುತ್ತದೆ. ಇದು Foundry Local ನಲ್ಲಿ ನಡೆಯುವ ಮಾದರಿಗಳ (ಉದಾ., **Phi-4-mini**) ಸಂಪರ್ಕವನ್ನು ಪ್ರದರ್ಶಿಸುತ್ತದೆ, ಸ್ವಯಂಚಾಲಿತ ಮಾದರಿ ಪತ್ತೆ ಮಾಡಿ, ಕ್ಲೌಡ್ ಸೇವೆಗಳಿಗೆ ಅವಲಂಬಿಸದೇ AI ಅನ್ವಯಿಕೆಗಳನ್ನು ಚಾಲನೆ ಮಾಡಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ.

### ಪೆಟ್ ಸ್ಟೋರಿ ಜೆನರೆಟರ್

**[ಪೆಟ್ ಸ್ಟೋರಿ ಜೆನರೆಟರ್](petstory/README.md)** ಒಂದು ಆಕರ್ಷಕ Spring Boot ವೆಬ್ ಅನ್ವಯಿಕೆ, ಇದು **ಬಹು-ಮೋಡಲ್ AI ಪ್ರಕ್ರಿಯೆ** ಬಳಸಿ ಸೃಜನಶೀಲ ಪೆಟ್ ಕಥೆಗಳನ್ನು ಸೃಷ್ಟಿಸುವುದನ್ನು ತೋರಿಸುತ್ತದೆ. ಇದು ಕ್ಲೈಂಟ್‌-ಸೈಡ್ ಮತ್ತು ಸರ್ವರ್-ಸೈಡ್ AI ಸಾಮರ್ಥ್ಯಗಳನ್ನು transformer.js ನೊಂದಿಗೆ ಬ್ರೌಸರ್ ಆಧಾರಿತ AI ಸಂವಹನಕ್ಕಾಗಿ ಮತ್ತು OpenAI SDK ಮೂಲಕ ಸರ್ವರ್-ಸೈಡ್ ಪ್ರಕ್ರಿಯೆಗಾಗಿ ಸಂಯೋಜಿಸುತ್ತದೆ.

### MCP ಕ್ಯಾಲಕುಲೇಟರ್ ಸೇವೆ (ಶುಭಾರಂಭ MCP ಡೆಮೋ)

**[MCP ಕ್ಯಾಲಕುಲೇಟರ್ ಸೇವೆ](calculator/README.md)** Spring AI ಉಪಯೋಗಿಸಿ **ಮಾದರಿ ಸಂತಕ ಮೇಳ (MCP)** ನ ಸರಳ ಪ್ರದರ್ಶನವಾಗಿದೆ. ಇದು MCP ಸಂಕ್ಷಿಪ್ತ ಪರಿಚಯವನ್ನು ನೀಡುತ್ತದೆ, MCP ಕ್ಲೈಂಟ್‌ಗಳೊಂದಿಗೆ ಸಂವಹನ ಮಾಡುವ ಮೂಲಭೂತ MCP ಸೆರ್ವರ್ ಅನ್ನು ಹೇಗೆ ರಚಿಸುವುದು ತೋರಿಸುತ್ತದೆ.

## ಕಲಿಕೆ ಪ್ರಗತಿ

ಈ ಪ್ರಾಜೆಕ್ಟ್‌ಗಳನ್ನು ಹಿಂದಿನ ಅಧ್ಯಾಯದ ಧಾರಣೆಗಳ ಮೇಲೆ ನಿರ್ಮಿಸಿರುವಂತೆ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ:

1. **ಸರಳವಾಗಿ ಪ್ರಾರಂಭಿಸಿ**: Foundry Local Spring Boot ಡೆಮೋದಿಂದ ಪ್ರಾರಂಭಿಸಿ ಸ್ಥಳೀಯ ಮಾದರಿಗಳೊಂದಿಗೆ ಮೂಲ AI ಇಂಟಿಗ್ರೇಶನನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಿ
2. **ಇಂಟರಾಕ್ಟಿವಿಟಿ ಸೇರಿಸಿ**: ಬಹು-ಮೋಡಲ್ AI ಮತ್ತು ವೆಬ್ ಆಧಾರಿತ ಸಂವಹನಗಳಿಗೆ ಪೆಟ್ ಸ್ಟೋರಿ ಜೆನರೆಟರ್‌ಗೆ ಮುಂದುವರಿಯಿರಿ
3. **MCP ಮೂಲತತ್ವಗಳನ್ನು ಕಲಿಯಿರಿ**: Model Context Protocol ಮೂಲಭೂತಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು MCP ಕ್ಯಾಲಕುಲೇಟರ್ ಸೇವೆಯನ್ನು ಪ್ರಯತ್ನಿಸಿ

## ಸಾರಾಂಶ

ಸುಂದರ ಕೆಲಸ! ನೀವು ಈಗ ಕೆಲವು ನಿಜವಾದ ಅನ್ವಯಿಕೆಗಳನ್ನು ಪರಿಶೀಲಿಸಿದ್ದೀರಿ:

- ಬ್ರೌಸರ್ ಮತ್ತು ಸರ್ವರ್ ಎರಡಲ್ಲಿಯೂ ಕಾರ್ಯನಿರ್ವಹಿಸುವ ಬಹು-ಮೋಡಲ್ AI ಅನುಭವಗಳು  
- ಆಧುನಿಕ ಜಾವಾ ಫ್ರೆಮ್ವರ್ಕ್‌ಗಳು ಮತ್ತು SDK ಗಳೊಂದಿಗೆ ಸ್ಥಳೀಯ AI ಮಾದರಿ ಇಂಟಿಗ್ರೇಶನ್  
- ನಿಮ್ಮ ಮೊದಲ Model Context Protocol ಸೇವೆ, ಎಐೊಂದಿಗೆ ಸಾಧನಗಳು ಹೇಗೆ ಸಂಯೋಜಿಸುವುದನ್ನು ನೋಡುವದು
