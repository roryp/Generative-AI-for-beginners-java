## מה תלמדו
בפרק זה נציג שלושה יישומים מעשיים המדגימים דפוסי פיתוח של בינה מלאכותית גנרטיבית עם Java:
- יצירת מחולל סיפורי חיות מחמד רב-מודאלי המשלב בינה מלאכותית בצד הלקוח ובצד השרת
- מימוש אינטגרציה עם מודל בינה מלאכותית מקומי באמצעות הדמו של Foundry Local Spring Boot
- פיתוח שירות פרוטוקול הקשר מודל (MCP) באמצעות דוגמת מחשבון

## הקדמה

פרק זה מציג **פרויקטים לדוגמה** המדגימים דפוסי פיתוח של בינה מלאכותית גנרטיבית עם Java. כל פרויקט פונקציונלי לחלוטין ומציג טכנולוגיות בינה מלאכותית מסוימות, דפוסי ארכיטקטורה ונהלים מומלצים שניתן לאמץ ליישומים שלכם.

### דמו Foundry Local Spring Boot

**[דמו Foundry Local Spring Boot](foundrylocal/README.md)** מדגים כיצד להתממשק עם מודלים של בינה מלאכותית מקומיים באמצעות **OpenAI Java SDK**. הוא מראה חיבור למודלים הפועלים על Foundry Local (למשל **Phi-4-mini**), עם זיהוי אוטומטי של מודלים, המאפשר להריץ יישומי בינה מלאכותית בלי להסתמך על שירותי ענן.

### מחולל סיפורי חיות מחמד

**[מחולל סיפורי חיות מחמד](petstory/README.md)** הוא יישום רשת Spring Boot מרתק המדגים **עיבוד בינה מלאכותית רב-מודאלי** ליצירת סיפורים יצירתיים על חיות מחמד. הוא משלב יכולות בינה מלאכותית בצד הלקוח ובצד השרת באמצעות transformer.js לאינטראקציה בדפדפן ו-OpenAI SDK לעיבוד בצד השרת.

### שירות מחשבון MCP (דמו MCP ידידותי למתחילים)

**[שירות מחשבון MCP](calculator/README.md)** הוא דמו פשוט של **פרוטוקול הקשר מודל (MCP)** המשתמש ב-Spring AI. הוא מספק מבוא ידידותי למתחילים למושגי MCP, ומדגים כיצד ליצור שרת MCP בסיסי המתקשר עם לקוחות MCP.

## התקדמות בלמידה

הפרויקטים האלה נועדו לבנות על מושגים מהפרקים הקודמים:

1. **התחל בפשטות**: התחל עם הדמו של Foundry Local Spring Boot להבנת אינטגרציה בסיסית עם מודלים מקומיים
2. **הוסף אינטראקטיביות**: המשך למחולל סיפורי חיות מחמד לאינטראקציות רשת ולבינה מלאכותית רב-מודאלית
3. **למד את יסודות MCP**: נסה את שירות מחשבון MCP להבנת יסודות פרוטוקול הקשר מודל

## סיכום

עבודה יפה! עתה חקרת כמה יישומים ממשיים:

- חוויות בינה מלאכותית רב-מודאליות שעובדות גם בדפדפן וגם בשרת
- אינטגרציה עם מודל בינה מלאכותית מקומי באמצעות מסגרות Java מודרניות ו-SDKs
- שירות פרוטוקול הקשר מודל ראשון לחוות כיצד כלים משתלבים עם בינה מלאכותית
