## Τι Θα Μάθετε
Σε αυτήν την ενότητα θα παρουσιάσουμε τρεις πρακτικές εφαρμογές που δείχνουν πρότυπα ανάπτυξης γενετικής AI με Java:
- Δημιουργία πολυτροπικού γεννήτριας ιστοριών με κατοικίδια συνδυάζοντας AI στην πλευρά πελάτη και στην πλευρά διακομιστή

- Ανάπτυξη υπηρεσίας Πρωτοκόλλου Συμφραζόμενου Μοντέλου (MCP) με το παράδειγμα του Αριθμομηχανή

## Εισαγωγή

Αυτή η ενότητα παρουσιάζει **δείγματα έργων** που επιδεικνύουν πρότυπα ανάπτυξης γενετικής AI με Java. Κάθε έργο είναι πλήρως λειτουργικό και παρουσιάζει συγκεκριμένες τεχνολογίες AI, αρχιτεκτονικά πρότυπα και βέλτιστες πρακτικές που μπορείτε να προσαρμόσετε για τις δικές σας εφαρμογές.

### Επίδειξη Foundry Local Spring Boot

Η **[Επίδειξη Foundry Local Spring Boot](foundrylocal/README.md)** παρουσιάζει πώς να ενσωματωθείτε με τοπικά μοντέλα AI χρησιμοποιώντας το **OpenAI Java SDK**. Επιδεικνύει τη σύνδεση με μοντέλα που τρέχουν στο Foundry Local (π.χ., **Phi-4-mini**), με αυτόματη ανίχνευση μοντέλων, επιτρέποντάς σας να εκτελείτε εφαρμογές AI χωρίς να βασίζεστε σε υπηρεσίες cloud.

### Γεννήτρια Ιστοριών με Κατοικίδια

Η **[Γεννήτρια Ιστοριών με Κατοικίδια](petstory/README.md)** είναι μια διαδραστική εφαρμογή web Spring Boot που επιδεικνύει **πολυτροπική επεξεργασία AI** για τη δημιουργία δημιουργικών ιστοριών με κατοικίδια. Συνδυάζει τις δυνατότητες AI στην πλευρά πελάτη και στην πλευρά διακομιστή χρησιμοποιώντας transformer.js για αλληλεπιδράσεις AI στο πρόγραμμα περιήγησης και το OpenAI SDK για επεξεργασία στην πλευρά διακομιστή.

### Υπηρεσία Αριθμομηχανής MCP (Επίδειξη MCP για Αρχάριους)

Η **[Υπηρεσία Αριθμομηχανής MCP](calculator/README.md)** είναι μια απλή επίδειξη του **Πρωτοκόλλου Συμφραζόμενου Μοντέλου (MCP)** χρησιμοποιώντας το Spring AI. Παρέχει μια φιλική προς αρχάριους εισαγωγή στις έννοιες MCP, δείχνοντας πώς να δημιουργήσετε έναν βασικό MCP Server που αλληλεπιδρά με MCP πελάτες.

## Πρόοδος Μάθησης

Αυτά τα έργα έχουν σχεδιαστεί ώστε να χτίζουν πάνω σε έννοιες από προηγούμενα κεφάλαια:

1. **Ξεκινήστε Απλά**: Ξεκινήστε με την Επίδειξη Foundry Local Spring Boot για να κατανοήσετε την βασική ενσωμάτωση AI με τοπικά μοντέλα
2. **Προσθέστε Διαδραστικότητα**: Προχωρήστε στη Γεννήτρια Ιστοριών με Κατοικίδια για πολυτροπική AI και διαδικτυακές αλληλεπιδράσεις
3. **Μάθετε τα Βασικά του MCP**: Δοκιμάστε την Υπηρεσία Αριθμομηχανής MCP για να κατανοήσετε τα βασικά του Πρωτοκόλλου Συμφραζόμενου Μοντέλου

## Περίληψη

Μπράβο! Τώρα έχετε εξερευνήσει μερικές πραγματικές εφαρμογές:

- Εμπειρίες πολυτροπικής AI που λειτουργούν τόσο στο πρόγραμμα περιήγησης όσο και στον διακομιστή
- Τοπική ενσωμάτωση μοντέλου AI χρησιμοποιώντας σύγχρονα Java frameworks και SDKs
- Την πρώτη σας υπηρεσία Πρωτοκόλλου Συμφραζόμενου Μοντέλου για να δείτε πώς τα εργαλεία ενσωματώνονται με το AI
