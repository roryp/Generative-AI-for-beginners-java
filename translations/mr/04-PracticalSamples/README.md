## तुम्ही काय शिकणार आहात
या विभागात आम्ही तीन व्यावहारिक अनुप्रयोगांचे डेमो दाखवणार आहोत जे Java सह जनरेटिव्ह AI विकास नमुने दर्शवितात:
- क्लायंट-साइड आणि सर्व्हर-साइड AI एकत्र करून बहु-मोडल पाळीव प्राणी कथा जनरेटर तयार करा
- Foundry Local Spring Boot डेमोसह स्थानिक AI मॉडेल एकत्रीकरण लागू करा
- कॅल्क्युलेटर उदाहरणासह मॉडेल संदर्भ प्रोटोकॉल (MCP) सेवा विकसित करा

## परिचय

हा अध्याय Java सह जनरेटिव्ह AI विकास नमुने दाखवणारे **नमुना प्रकल्प** सादर करतो. प्रत्येक प्रकल्प पूर्णपणे कार्यक्षम आहे आणि विशिष्ट AI तंत्रज्ञान, आर्किटेक्चर पॅटर्न्स, आणि सर्वोत्तम पद्धती दाखवतो ज्यांना तुम्ही तुमच्या स्वतःच्या अनुप्रयोगांसाठी अनुकूलित करू शकता.

### Foundry Local Spring Boot डेमो

**[Foundry Local Spring Boot डेमो](foundrylocal/README.md)** हे स्थानिक AI मॉडेल्ससह कसे एकत्रित करायचे हे दर्शविते, **OpenAI Java SDK** वापरून. हे Foundry Local वर चालणाऱ्या मॉडेल्सशी (उदाहरणार्थ, **Phi-4-mini**) जोडणी दाखवते, स्वयंचलित मॉडेल शोधणीसह, ज्यामुळे तुम्ही क्लाउड सेवांवर अवलंबून न राहता AI अनुप्रयोग चालवू शकता.

### पाळीव प्राणी कथा जनरेटर

**[पाळीव प्राणी कथा जनरेटर](petstory/README.md)** एक आकर्षक Spring Boot वेब अनुप्रयोग आहे जो **बहु-मोडल AI प्रोसेसिंग** दाखवतो, जे सर्जनशील पाळीव प्राणी कथा तयार करतो. हे क्लायंट-साइड आणि सर्व्हर-साइड AI क्षमता एकत्रित करते, ज्यात ब्राउझर-आधारित AI संवादांसाठी transformer.js आणि सर्व्हर-साइड प्रोसेसिंगसाठी OpenAI SDK वापरले आहे.

### MCP कॅल्क्युलेटर सेवा (सोप्या वापरासाठी MCP डेमो)

**[MCP कॅल्क्युलेटर सेवा](calculator/README.md)** ही Spring AI वापरून **मॉडेल संदर्भ प्रोटोकॉल (MCP)** चे एक साधे डेमो आहे. हे MCP संकल्पनांसाठी सोपा परिचय देते, कसे एक मूलभूत MCP सर्व्हर तयार करायचा ज्याचा संवाद MCP क्लायंट्स सोबत होतो, हे दाखवते.

## शिकण्याची प्रगती

हे प्रकल्प मागील अध्यायांमधील संकल्पनांवर आधारित तयार केले आहेत:

1. **सोप्या पासून सुरुवात करा:** Foundry Local Spring Boot डेमोपासून स्थानिक मॉडेल्ससह मूलभूत AI एकत्रीकरण समजून घ्या  
2. **परस्परसंवाद जोडा:** बहु-मोडल AI आणि वेब-आधारित संवादांसाठी पाळीव प्राणी कथा जनरेटर कडे प्रगती करा  
3. **MCP मूलतत्त्वे शिका:** मॉडेल संदर्भ प्रोटोकॉलच्या मूलभूत गोष्टी समजून घेण्याकरिता MCP कॅल्क्युलेटर सेवा वापरा  

## सारांश

छान काम! तुम्ही आता काही वास्तविक अनुप्रयोगांची माहिती घेतली आहे:

- ब्राउझर आणि सर्व्हर दोन्हीवर चालणारी बहु-मोडल AI अनुभव
- आधुनिक Java फ्रेमवर्क आणि SDK वापरून स्थानिक AI मॉडेल एकत्रीकरण
- तुमची पहिली मॉडेल संदर्भ प्रोटोकॉल सेवा जिथे AI आणि टूल्स एकत्र काम करतात
