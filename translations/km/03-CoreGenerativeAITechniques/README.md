## бһ”бһүбҹ’бһҮбһёбһҳбһ¶бһҸбһ·бһҖбһ¶

- [бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ](#бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ)

  - [бһҮбҹҶбһ бһ¶бһ“ 2: бһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҗбһҸбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ](#бһҮбҹҶбһ бһ¶бһ“-2-бһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҗбһҸбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ)
- [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһјбһҢбҹӮбһӣ](#бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһјбһҢбҹӮбһӣ)
- [бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 1: бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶ LLM](#бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё-1-бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶-llm)
- [бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 2: бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ](#бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё-2-бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ)
- [бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 3: RAG (Retrieval-Augmented Generation)](#бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё-3-rag-retrieval-augmented-generation)
- [бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 4: AI бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһң](#бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё-4-ai-бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһң)
- [бһӣбҹҶбһ“бһ¶бҹҶбһ‘бһјбһ‘бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ—](#бһӣбҹҶбһ“бһ¶бҹҶбһ‘бһјбһ‘бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ—)
- [бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ](#бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ)
- [бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶](#бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶)
  - [бһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…](#бһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…)


## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹ„бһҷбһҠбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ”бһ„бҹ’бһҖбһҫбһҸ AI бһӮбҹ’бһҡбһ№бҹҮбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Java бһ“бһ·бһ„ GitHub Modelsбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбһ¶бһ бһҡбһҺбҹҲбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLMs), бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ, бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ RAG (retrieval-augmented generation), бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһўбһ’бһ·бһ”бҹ’бһ”бһ¶бһҷ AI бһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбҹ”

## бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҮбһ¶бһҳбһ»бһ“

бһҳбһ»бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бҹ–
- Java 21 бһ¬бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһ“бҹҒбҹҮбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„
- Maven бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҜбһҖбһҹбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷ
- бһӮбһҺбһ“бһё GitHub бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„ token бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ (PAT)

## бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһјбһҢбҹӮбһӣ

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбҹӮбһӣбһ“бһ¶бһ“бһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҡбһҪбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбҹӮбһҳбһҪбһҷбҹ—бҹ–

**GPT-4.1-nano** (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢ):
- бһӣбһҝбһ“бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ“бһ·бһ„бһҗбҹ’бһӣбҹғбһҸбһ·бһ…бһҒбҹ’бһӣбһ¶бҹҶбһ„
- бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһҹбһ“бҹ’бһ‘бһ“бһ¶
- бһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҖбһ“бһӣбҹҶбһ“бһ¶бҹҶбһ”бһүбҹ’бһ…бһјбһӣ LLM бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

**GPT-4o-mini** (бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһ»бһҒбһ„бһ¶бһҡ, RAG, бһ“бһ·бһ„ AI бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһң):
- бһҳбҹүбһјбһҢбҹӮбһӣбһҸбһјбһ…бһҸбҹӮбһ–бҹҒбһүбһӣбҹҒбһүбһ“бһјбһңбһўбһ“бһ»бһӮбҹ’бһҡбҹ„бҹҮ "бһҮбҹҶбһ“бһҪбһҷбһӮбҹ’бһҡбһ”бҹӢбһңбһ·бһҹбҹҗбһҷ"
- бһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢбһ–бһёбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ–
  - бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҡбһјбһ”бһ—бһ¶бһ–
  - бһ”бһүбҹ’бһ…бһјбһӣ JSON/бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’
  - бһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ/бһҳбһ»бһҒбһ„бһ¶бһҡ
- бһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„ nano бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҮбһ¶бһ“бһ·бһ…бҹ’бһ…

> **бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢ**бҹ– бһ”бһҫбһ‘бҹ„бҹҮбһ”бһёбһҳбҹүбһјбһҢбҹӮбһӣ "nano" бһӣбһҝбһ“бһ“бһ·бһ„бһҗбҹ’бһӣбҹғбһҗбҹ„бһҖбһҖбҹҸбһҠбҹ„бһҷ бһҳбҹүбһјбһҢбҹӮбһӣ "mini" бһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ”бҹ’бһҡбһҹбһҫбһҡбһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ бһҠбҹӮбһӣбһ”бҹ’бһҡбһ бҹӮбһӣбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҒбһүбһӣбҹҒбһүбһҠбҹ„бһҷбһўбҹ’бһ“бһҖбһ•бҹ’бһҠбһӣбҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҢбҹӮбһӣ nano бһ‘бҹҒбҹ”

### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҡбҹҖбһ“

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһӮбҹ’бһҡбһ№бҹҮбһ“бҹғбһҡбһ”бҹҖбһ”бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҢбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLM) бһҸбһ¶бһҳбһҡбһҷбҹҲ OpenAI API бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ client бһҮбһ¶бһҳбһҪбһҷ GitHub Models бһӣбҹҶбһ“бһ¶бҹҶбһҹбһ¶бһҡ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ’бһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбһ“бҹ’бһ‘бһ“бһ¶ бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһңбҹӮбһ„бһ“бһ·бһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбһ“бҹғбһ…бһҳбҹ’бһӣбһҫбһҷбҹ”

### бһҷбһӣбҹӢбһҠбһ№бһ„бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ–бһёбһҖбһјбһҠ

#### 1. бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ client
```java
// бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбһ·бһҗбһ·бһҮбһ“ AI
OpenAIClient client = new OpenAIClientBuilder()
    .endpoint("https://models.inference.ai.azure.com")
    .credential(new StaticTokenCredential(pat))
    .buildClient();
```

бһ“бҹҒбҹҮбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ… GitHub Models бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ token бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

#### 2. бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһҹбһ¶бһҳбһүбҹ’бһү
```java
List<ChatRequestMessage> messages = List.of(
    // бһҹбһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбҹҶбһҺбһҸбҹӢбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶ AI
    new ChatRequestSystemMessage("You are a helpful Java expert."),
    // бһҹбһ¶бһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһ¶бһ“бһҹбҹҶбһҺбһҪбһҡбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ
    new ChatRequestUserMessage("Explain Java streams briefly.")
);

ChatCompletionsOptions options = new ChatCompletionsOptions(messages)
    .setModel("gpt-4.1-nano")  // бһҳбҹүбһјбһҢбҹӮбһӣбһӣбһҝбһ“ бһ“бһ·бһ„бһҗбҹ’бһӣбҹғбһҸбһ·бһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
    .setMaxTokens(200)         // бһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһңбҹӮбһ„бһ…бһҳбҹ’бһӣбһҫбһҷ
    .setTemperature(0.7);      // бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бһ¶бһ–бһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ (0.0-1.0)
```

#### 3. бһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһҹбһ“бҹ’бһ‘бһ“бһ¶
```java
// бһ”бһ“бҹ’бһҗбҹӮбһҳбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһё AI бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ“бҹ’бһ‘бһ“бһ¶
messages.add(new ChatRequestAssistantMessage(aiResponse));
messages.add(new ChatRequestUserMessage("Follow-up question"));
```

AI бһ“бһ№бһ„бһ…бһ„бһ…бһ¶бҹҶбһҹбһ¶бһҡвҖӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһҳбһ»бһ“бһҳбҹ’бһҸбһ„бһ‘бҹ„бҹҮбһ”бһёбһҸбҹ’бһҡбһјбһңбһ”бһүбҹ’бһ…бһјбһӣбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһҫбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҖбҹҸбһҠбҹ„бһҷбҹ”

1. **бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢбһҹбһ¶бһҳбһүбҹ’бһү**бҹ– AI бһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡбһ–бһё Java бһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

3. **бһҹбһ“бҹ’бһ‘бһ“бһ¶бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһ¶бһ“**бҹ– бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ–бһ·бһҸбһҮбһ¶бһҳбһҪбһҷ AI

### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҡбҹҖбһ“

бһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҳбҹүбһјбһҢбҹӮбһӣ AI бһҹбҹ’бһ“бһҫбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһ’бһҳбҹ’бһҳбһҸбһ¶бһҖбҹ’бһҡбҹ… бһ“бһ·бһ„ APIs бһҸбһ¶бһҳбһҡбһҷбҹҲбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶ бһҠбҹӮбһӣбһҳбҹүбһјбһҢбҹӮбһӣбһўбһ¶бһ“бһҹбҹҶбһҺбһҫбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ· бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ JSON Schema бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһ·бһӣбһҸбһ”бһҸбһ¶бһҳбһӣбһ‘бҹ’бһ’бһ•бһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бһҡбһ·бһ”бһ‘ бһҒбһҺбҹҲбһҠбҹӮбһӣбһҖбҹҶбһ бһ»бһҹбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳбһҸбһ¶бһҳбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бһ·бһ„бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бҹ”

> **бһ…бҹҶбһҺбһ¶бҹҶ**бҹ– бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫ `gpt-4o-mini` бһ–бһёбһ–бҹ’бһҡбҹ„бҹҮбһҖбһ¶бһҡбһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбҹҶбһ“бһ¶бһүбһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһ¶бһ“бһ—бһ¶бһ–бһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ бһҠбҹӮбһӣбһ”бҹ’бһҡбһ бҹӮбһӣбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҒбһүбһӣбҹҒбһүбһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһјбһҢбҹӮбһӣ nano бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ”

    messages.add(new ChatRequestToolMessage(result, toolCall.getId()));

    ChatCompletions finalResponse = client.getChatCompletions(MODEL, options);

1. **бһҳбһ»бһҒбһ„бһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»**бҹ– AI бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Seattle, бһўбҹ’бһ“бһҖбһ•бҹ’бһҸбһӣбҹӢбһңбһ¶, AI бһҡбҹҖбһ”бһ…бҹҶбһ…бһҳбҹ’бһӣбһҫбһҷ
2. **бһҳбһ»бһҒбһ„бһ¶бһҡбһӮбһҺбһ“бһ¶**бҹ– AI бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһӮбһҺбһ“бһ¶бһҖбһ¶бһҡ (15% бһ“бҹғ 240), бһўбҹ’бһ“бһҖбһӮбһҺбһ“бһ¶, AI бһ–бһ“бҹ’бһҷбһӣбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣ

## бһҳбҹҒбһҡбҹҖбһ“бһ‘бһё 3: RAG (Retrieval-Augmented Generation)

**бһҜбһҖбһҹбһ¶бһҡ:** `src/main/java/com/example/genai/techniques/rag/SimpleReaderDemo.java`

### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҡбҹҖбһ“

Retrieval-Augmented Generation (RAG) бһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһҷбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҳбһҖбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһҹбһ¶ бһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһјбһӣбһ”бһҡбһ·бһ”бһ‘бһҜбһҖбһҹбһ¶бһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһҫ AI бһ бһҫбһҷбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҳбҹүбһјбһҢбҹӮбһӣбһўбһ¶бһ…бһ•бҹ’бһҸбһӣбҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бҹ’бһҡбһ—бһ–бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҮбһ¶бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҠбһ»бҹҮбһ”бһҺбҹ’бһҠбһ¶бһӣбһҠбҹӮбһӣбһ”бҹ’бһҡбһ бҹӮбһӣбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷ бһ¬бһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһңбҹҮбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҒбһҺбҹҲбһҡбһҖбҹ’бһҹбһ¶бһӮбҹ’бһҡбҹ„бһ„бһҒбҹ’бһҹбҹӮбһ”бһүбҹ’бһҮбһ¶ бһ…бҹҶбһ–бҹ„бҹҮбһҹбҹҶбһҺбһҪбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бһ–бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ‘бһјбһ‘бҹ…бһҸбһ¶бһҳбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбҹ’бһ”бһ·бһ…бһҶбһҳбһ¶бһҹбҹ”

> **бһ…бҹҶбһҺбһ¶бҹҶ**бҹ– бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫ `gpt-4o-mini` бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бһјбһңбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ prompting бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҡбһ·бһ”бһ‘бһҜбһҖбһҹбһ¶бһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҹбһҳбҹ’бһҡбҹҒбһ…бһҸбҹ’бһҡбһ„бҹӢ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ RAG бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбҹ”

#### 2. бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ”бһҡбһ·бһ”бһ‘

```java
private boolean isRefusalResponse(String response) {
    String lowerResponse = response.toLowerCase();
    String[] refusalPatterns = {
        "i can't assist with", "i cannot assist with",
        "sorry, i can't", "sorry, i cannot",
        "i'm unable to", "against my guidelines"
    };
    
    for (String pattern : refusalPatterns) {
        if (lowerResponse.contains(pattern)) {
            return true;
        }
    }
    return false;
}
```

### бһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…

**"GITHUB_TOKEN бһҳбһ·бһ“бһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ"**
- бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ environment variable бһҡбһҪбһ…бһ бһҫбһҷ
- бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶ token бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“ `models:read` scope

**"бһӮбҹ’бһҳбһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһё API"**
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҗбһ¶ token бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбһ»бһ–бһӣбһ—бһ¶бһ–
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҲбһ¶бһ“бһҠбһӣбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһҺбһҸбҹӢбһўбһҸбһ·бһ”бһҡбһҳбһ¶ (rate limits) бһ¬бһ“бҹ…

**бһҖбҹҶбһ бһ»бһҹбһ”бһүбҹ’бһ…бһјбһӣбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Maven**
- бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“ Java 21 бһ¬бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһ“бҹҒбҹҮ
- бһҡбһҸбҹӢ `mvn clean compile` бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҜбһҖбһҹбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷ
