## អ្វីដែលអ្នកនឹងរៀន
ក្នុងផ្នែកនេះយើងនឹងបង្ហាញពីកម្មវិធីប្រើប្រាស់ជាក់ស្តែងបី ដែលបង្ហាញពីលំនាំអភិវឌ្ឍន៍ AI ច្នៃប្រឌិតជាមួយ Java៖
- បង្កើតកម្មវិធីបង្កើតរឿងសត្វចិញ្ចឺម២ប្រភេទប្រើ AI ភាគបណ្ដាញ និងផ្នែកម៉ាស៊ីនមេ
- អនុវត្តការរួមបញ្ចូលម៉ូដែល AI ក្នុងកន្លែងជាមួយនឹងការបង្ហាញ Foundry Local Spring Boot
- អភិវឌ្ឍសេវាកម្ម Model Context Protocol (MCP) ជាមួយឧទាហរណ៍គណនាគ្រាប់

## ការណែនាំ

ជំពូកនេះបង្ហាញពី **គំរូគម្រោង** ដែលបង្ហាញលំនាំអភិវឌ្ឍន៍ AI ច្នៃប្រឌិតជាមួយ Java។ គម្រោងនីមួយៗមានមុខងារពេញលេញ ហើយបង្ហាញបច្ចេកវិទ្យា AI ជាក់លាក់ លំនាំស្ថាបត្យកម្ម និងអនុវត្តិវិធីល្អៗដែលអ្នកអាចដាក់បន្ថែមក្នុងកម្មវិធីរបស់អ្នក។

### ការបង្ហាញ Foundry Local Spring Boot

**[ការបង្ហាញ Foundry Local Spring Boot](foundrylocal/README.md)** បង្ហាញពីរបៀបរួមបញ្ចូលជាមួយម៉ូដែល AI ក្នុងកន្លែងប្រើ **OpenAI Java SDK**។ វាបង្ហាញពីការតភ្ជាប់ទៅម៉ូដែលរត់លើ Foundry Local (ឧ. **Phi-4-mini**), ជាមួយនឹងការរកម៉ូដែលដោយស្វ័យប្រវត្តិ ដែលអនុញ្ញាតឱ្យអ្នករត់កម្មវិធី AI ដោយមិនពឹងផ្អែកលើសេវាកម្មពពក។

### កម្មវិធីបង្កើតរឿងសត្វចិញ្ចឹម

**[កម្មវិធីបង្កើតរឿងសត្វចិញ្ចឹម](petstory/README.md)** ជាកម្មវិធី Spring Boot លើបណ្តាញដែលរីករាយ ហើយបង្ហាញពី **ការលេង AI ច្រើនប្រភេទ** ដើម្បីបង្កើតរឿងសត្វចិញ្ចឹមច្នៃប្រឌិត។ វារួមបញ្ចូលកម្លាំង AI ភាគបណ្ដាញ និងផ្នែកម៉ាស៊ីនមេ ប្រើ transformer.js សម្រាប់អន្តរកម្ម AI នៅក្នុងកម្មវិThan់ផ្ទាល់ និង OpenAI SDK សម្រាប់ដំណើរការផ្នែកម៉ាស៊ីនមេ។

### សេវាកម្ម MCP Calculator (ការបង្ហាញ MCP សាមញ្ញសម្រាប់អ្នកចាប់ផ្ដើម)

**[សេវាកម្ម MCP Calculator](calculator/README.md)** ជាការបង្ហាញសាមញ្ញនៃ **Model Context Protocol (MCP)** ប្រើ Spring AI។ វាប្រកបដោយការណែនាំសាមញ្ញអំពីគំនិត MCP បង្ហាញពីរបៀបបង្កើតម៉ាស៊ីនមេ MCP គន្លងមួយដែលអាចអន្តរកម្មជាមួយអតិថិជន MCP។

## មុខងាររៀន

គម្រោងទាំងនេះត្រូវបានរចនាឡើងដើម្បីបង្កើតលើមូលដ្ឋានគំនិតពីជំពូកមុនៗ៖

1. **ចាប់ផ្ដើមពីសាមញ្ញ**៖ ចាប់ផ្ដើមជាមួយការបង្ហាញ Foundry Local Spring Boot ដើម្បីយល់ពីការរួមបញ្ចូល AI មូលដ្ឋានជាមួយម៉ូដែលក្នុងកន្លែង
2. **បន្ថែមអន្តរកម្ម**៖ បន្តទៅកម្មវិធីបង្កើតរឿងសត្វចិញ្ចឹមសម្រាប់ AI ច្រើនប្រភេទ និងអន្តរកម្មលើបណ្តាញ
3. **រៀនគ្រឹះ MCP**៖ អ្នកអាចសាកល្បងសេវាកម្ម MCP Calculator ដើម្បីយល់ពីមូលដ្ឋាន Model Context Protocol

## សេចក្តីសង្ខេប

ការងារល្អ! ឥឡូវនឹងអ្នកបានស្ទើរតែនឹងបញ្ចូលកម្មវិធីពិតប្រាកដ៖

- បទពិសោធន៍ AI ច្រើនប្រភេទដែលដំណើរការរួមទាំងក្នុងកម្មវិធីវីប និងម៉ាស៊ីនមេ
- ការរួមបញ្ចូលម៉ូដែល AI ក្នុងកន្លែងប្រើបណ្តាញ Java និង SDK សម័យទំនើប
- សេវាកម្ម Model Context Protocol ដំបូងរបស់អ្នក ដើម្បីមើលពីរបៀបឧបករណ៍រួមបញ្ចូលជាមួយ AI
