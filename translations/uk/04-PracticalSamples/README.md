## Вступ

Цей розділ демонструє **прикладні проєкти**, які ілюструють шаблони розробки генеративного ШІ з Java. Кожен проєкт повністю функціональний та демонструє конкретні технології ШІ, архітектурні шаблони та найкращі практики, які ви можете адаптувати для власних застосунків.

### Демонстрація Foundry Local Spring Boot

**[Демонстрація Foundry Local Spring Boot](foundrylocal/README.md)** показує, як інтегруватися з локальними моделями ШІ за допомогою **OpenAI Java SDK**. Вона демонструє підключення до моделей, що працюють у Foundry Local (наприклад, **Phi-4-mini**), з автоматичним визначенням моделі, що дозволяє запускати застосунки ШІ без використання хмарних сервісів.

### Генератор історій про домашніх тварин

**[Генератор історій про домашніх тварин](petstory/README.md)** — це інтерактивний веб-застосунок на Spring Boot, який демонструє **мультимодальну обробку ШІ** для створення творчих історій про тварин. Він поєднує клієнтські та серверні можливості ШІ, використовуючи transformer.js для взаємодії з ШІ у браузері та OpenAI SDK для обробки на сервері.

### Сервіс MCP калькулятора (демо MCP для початківців)

**[Сервіс MCP калькулятора](calculator/README.md)** — це просте демонстраційне застосування **Model Context Protocol (MCP)** з використанням Spring AI. Він пропонує дружнє для початківців введення в концепції MCP, показуючи, як створити базовий MCP-сервер, що взаємодіє з MCP-клієнтами.

## Прогрес навчання

Ці проєкти спроєктовані так, щоб послідовно розширювати знання з попередніх розділів:

1. **Почніть з простого:** Почніть з демонстрації Foundry Local Spring Boot, щоб зрозуміти базову інтеграцію ШІ з локальними моделями
2. **Додайте інтерактивність:** Перейдіть до генератора історій про домашніх тварин для мультимодального ШІ та взаємодії через веб
3. **Вивчіть основи MCP:** Спробуйте сервіс MCP калькулятора, щоб зрозуміти основи протоколу Model Context Protocol

## Підсумок

Чудова робота! Тепер ви ознайомилися з реальними застосунками:

- Мультимодальні ШІ-досвіди, що працюють як у браузері, так і на сервері
- Інтеграція локальних моделей ШІ з використанням сучасних Java-фреймворків та SDK
- Ваш перший сервіс Model Context Protocol, який показує, як інструменти інтегруються з ШІ
