## Introduction

Dis chapter dey show **sample projects** wey demonstrate generative AI development patterns wit Java. Every project dey fully functional and e dey show particular AI technologies, architectural patterns, and beta beta way wey you fit use for your own applications.

### Foundry Local Spring Boot Demo

Di **[Foundry Local Spring Boot Demo](foundrylocal/README.md)** na how to integrate wit local AI models using di **OpenAI Java SDK**. E dey show how to connect to models wey dey run for Foundry Local (like **Phi-4-mini**), wit automatic model detection, wey mean say you fit run AI applications without to depend on cloud services.

### Pet Story Generator

Di **[Pet Story Generator](petstory/README.md)** na one interesting Spring Boot web application wey show **multi-modal AI processing** to generate creative pet stories. E combine client-side and server-side AI power using transformer.js for browser-based AI interactions plus di OpenAI SDK for server-side processing.

### MCP Calculator Service (Beginner-Friendly MCP Demo)

Di **[MCP Calculator Service](calculator/README.md)** na simple demo of di **Model Context Protocol (MCP)** using Spring AI. E provide simple beginner-friendly introduction to MCP concepts, di way to create basic MCP Server wey fit interact wit MCP clients.

## Learning Progression

Dis projects dem design to build on top of concepts from previous chapters:

1. **Start Simple**: Begin wit di Foundry Local Spring Boot Demo to understand basic AI integration wit local models
2. **Add Interactivity**: Move go Pet Story Generator for multi-modal AI and web-based interactions
3. **Learn MCP Basics**: Try di MCP Calculator Service to understand Model Context Protocol fundamentals
