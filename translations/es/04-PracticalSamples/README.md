## Introducción

Este capítulo presenta **proyectos de ejemplo** que demuestran patrones de desarrollo de IA generativa con Java. Cada proyecto es completamente funcional y muestra tecnologías de IA, patrones arquitectónicos y mejores prácticas que puedes adaptar para tus propias aplicaciones.

### Demo Foundry Local Spring Boot

La **[Demo Foundry Local Spring Boot](foundrylocal/README.md)** demuestra cómo integrar modelos de IA locales usando el **OpenAI Java SDK**. Muestra la conexión a modelos ejecutándose en Foundry Local (por ejemplo, **Phi-4-mini**), con detección automática de modelos, permitiéndote ejecutar aplicaciones de IA sin depender de servicios en la nube.

### Generador de Historias de Mascotas

El **[Generador de Historias de Mascotas](petstory/README.md)** es una aplicación web atractiva con Spring Boot que muestra el procesamiento de IA **multimodal** para generar historias creativas sobre mascotas. Combina capacidades de IA del lado cliente y servidor usando transformer.js para interacciones de IA en el navegador y el SDK OpenAI para el procesamiento en servidor.

### Servicio de Calculadora MCP (Demo MCP para Principiantes)

El **[Servicio de Calculadora MCP](calculator/README.md)** es una demostración simple del **Protocolo de Contexto de Modelo (MCP)** usando Spring AI. Proporciona una introducción amigable a conceptos MCP, mostrando cómo crear un MCP Server básico que interactúa con clientes MCP.

## Progresión de Aprendizaje

Estos proyectos están diseñados para construir sobre conceptos de capítulos anteriores:

1. **Comenzar Simple**: Inicia con la Demo Foundry Local Spring Boot para entender la integración básica de IA con modelos locales
2. **Agregar Interactividad**: Continua con el Generador de Historias de Mascotas para IA multimodal e interacciones web
3. **Aprender Fundamentos MCP**: Prueba el Servicio de Calculadora MCP para entender los fundamentos del Protocolo de Contexto de Modelo
