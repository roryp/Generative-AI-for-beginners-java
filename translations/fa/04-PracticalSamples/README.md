[![برنامه‌ها و پروژه‌های عملی](https://img.youtube.com/vi/01vJsYei3H0/0.jpg)](https://www.youtube.com/watch?v=01vJsYei3H0 "برنامه‌ها و پروژه‌های عملی")

## آنچه خواهید آموخت
در این بخش، سه برنامه عملی را نمایش می‌دهیم که الگوهای توسعه هوش مصنوعی مولد را با جاوا نشان می‌دهند:
- ساخت یک ژنراتور داستان حیوان خانگی چندحالته که هوش مصنوعی سمت مشتری و سرور را ترکیب می‌کند
- پیاده‌سازی ادغام مدل هوش مصنوعی محلی با دمو Foundry Local Spring Boot
- توسعه یک سرویس پروتکل متن مدل (MCP) با مثال ماشین‌حساب

## مقدمه

این فصل **پروژه‌های نمونه** را نشان می‌دهد که الگوهای توسعه هوش مصنوعی مولد با جاوا را به نمایش می‌گذارند. هر پروژه کاملاً عملی است و فناوری‌های خاص هوش مصنوعی، الگوهای معماری و بهترین شیوه‌ها را که می‌توانید برای برنامه‌های خود تطبیق دهید، نشان می‌دهد.

### دموی Foundry Local Spring Boot

**[دموی Foundry Local Spring Boot](foundrylocal/README.md)** نشان می‌دهد چطور می‌توان با مدل‌های هوش مصنوعی محلی با استفاده از **کتابخانه Java SDK اپن‌ای‌آی** ادغام کرد. این دمو اتصال به مدل‌هایی که روی Foundry Local اجرا می‌شوند (مانند **Phi-4-mini**) را به همراه شناسایی خودکار مدل نمایش می‌دهد، که به شما امکان اجرای برنامه‌های هوش مصنوعی بدون اتکا به خدمات ابری را می‌دهد.

### ژنراتور داستان حیوان خانگی

**[ژنراتور داستان حیوان خانگی](petstory/README.md)**، یک برنامه وب به‌صورت Spring Boot جذاب است که **پردازش چندحالته هوش مصنوعی** برای تولید داستان‌های خلاقانه حیوانات خانگی را نشان می‌دهد. این برنامه قابلیت‌های هوش مصنوعی سمت مشتری و سرور را با استفاده از transformer.js برای تعامل‌های هوش مصنوعی مبتنی بر مرورگر و کتابخانه SDK اپن‌ای‌آی برای پردازش سمت سرور ترکیب می‌کند.

### سرویس ماشین‌حساب MCP (دموی MCP مناسب برای مبتدیان)

**[سرویس ماشین‌حساب MCP](calculator/README.md)**، یک نمونه ساده از **پروتکل متن مدل (MCP)** با استفاده از Spring AI است. این سرویس معرفی دوستانه‌ای برای مفاهیم MCP ارائه می‌دهد و نشان می‌دهد چگونه یک سرور MCP ساده بسازید که با کلاینت‌های MCP تعامل داشته باشد.

## فرآیند یادگیری

این پروژه‌ها طراحی شده‌اند تا مفاهیم فصل‌های قبلی را گسترش دهند:

1. **شروع ساده**: با دموی Foundry Local Spring Boot شروع کنید تا ادغام پایه‌ای هوش مصنوعی با مدل‌های محلی را بفهمید
2. **افزودن تعامل**: به سمت ژنراتور داستان حیوان خانگی حرکت کنید برای هوش مصنوعی چندحالته و تعاملات مبتنی بر وب
3. **یادگیری مبانی MCP**: سرویس ماشین‌حساب MCP را امتحان کنید تا اصول پروتکل متن مدل را درک کنید

## خلاصه

کار خوبی انجام دادید! حالا برخی برنامه‌های واقعی را بازدید کرده‌اید:

- تجربه‌های چندحالته هوش مصنوعی که در مرورگر و سرور کار می‌کنند
- ادغام مدل‌های هوش مصنوعی محلی با استفاده از فریم‌ورک‌ها و SDKهای مدرن جاوا
- اولین سرویس پروتکل متن مدل خود را برای درک چگونگی ادغام ابزارها با هوش مصنوعی تجربه کرده‌اید
