**所需時間**: 整個工作坊可於線上完成，無需本地安裝。環境設置約需 2 分鐘，探索範例則根據深度需時 1-3 小時不等。

> **快速開始**

1. 將此儲存庫 fork 到您的 GitHub 帳戶
2. 點擊 **Code** → **Codespaces** 分頁 → **...** → **New with options...**
3. 使用預設設定 — 此設定將選擇本課程專用之開發容器
4. 點擊 **Create codespace**
5. 等候約 2 分鐘以完成環境準備
6. 直接跳至 [第一個範例](./02-SetupDevEnvironment/README.md#step-2-create-a-github-personal-access-token)

> **偏好本地 Clone？**
>
> 本儲存庫含有超過 50 種語言翻譯，會大幅增加下載大小。若想不含翻譯的 Clone，請使用稀疏簽出（sparse checkout）：
>
> **Linux / macOS (Bash)**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/Generative-AI-for-beginners-java.git
> cd Generative-AI-for-beginners-java
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **Windows (PowerShell)**
> ```powershell
> git clone --filter=blob:none --sparse https://github.com/microsoft/Generative-AI-for-beginners-java.git
> cd Generative-AI-for-beginners-java
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
> 這樣您將只下載完成課程所需的內容，速度更快。

## 課程架構與學習路徑

### **第一章：生成式 AI 入門**
- **核心概念**：了解大型語言模型、詞元、詞嵌入及 AI 功能
- **Java AI 生態系統**：Spring AI 及 OpenAI SDK 簡介
- **模型上下文協定**：介紹 MCP 及其在 AI 代理人通訊中的角色
- **實務應用**：實際案例包括聊天機器人與內容生成
- **[→ 開始第一章](./01-IntroToGenAI/README.md)**

### **第二章：開發環境設置**
- **多供應商配置**：設定 GitHub 模型、Azure OpenAI 及 OpenAI Java SDK 整合
- **Spring Boot + Spring AI**：企業 AI 應用開發最佳實踐
- **GitHub 模型**：免費 AI 模型存取，適合原型設計與學習（無需信用卡）
- **開發工具**：Docker 容器、VS Code 與 GitHub Codespaces 設定
- **[→ 開始第二章](./02-SetupDevEnvironment/README.md)**

### **第三章：核心生成式 AI 技術**
- **提示工程**：優化 AI 模型回應的技巧
- **詞嵌入與向量操作**：實作語意搜尋及相似度比對
- **檢索增強生成 (RAG)**：結合 AI 與您自身的資料來源
- **函數調用**：透過自訂工具與插件擴展 AI 能力
- **[→ 開始第三章](./03-CoreGenerativeAITechniques/README.md)**

### **第四章：實務應用與專案**
- **寵物故事產生器** (`petstory/`)：使用 GitHub 模型進行創意內容生成
- **Foundry 本地示範** (`foundrylocal/`)：本地 AI 模型整合 OpenAI Java SDK
- **MCP 計算機服務** (`calculator/`)：Spring AI 實現基本模型上下文協定
- **[→ 開始第四章](./04-PracticalSamples/README.md)**

### **第五章：負責任的 AI 開發**
- **GitHub 模型安全性**：測試內建過濾與安全機制（硬性封鎖與軟性拒絕）
- **負責任 AI 示範**：實作現代 AI 安全系統的操作範例
- **最佳實務**：道德 AI 開發及部署的關鍵準則
- **[→ 開始第五章](./05-ResponsibleGenAI/README.md)**
