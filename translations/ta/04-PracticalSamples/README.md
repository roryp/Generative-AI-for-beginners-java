## நீங்கள் கற்பது என்ன
இந்த பிரிவில் நாங்கள் ஜாவாவுடன் உருவாக்கும் செயற்கை நுண்ணறிவு உருவாக்க வடிவமைப்புகளை காட்டும் மூன்று நடைமுறை பயன்பாடுகளை சுட்டிக் காண்பிப்போம்:
- கிளையன்ட்-சைடு மற்றும் சர்வர்-சைடு AI ஒன்றிணைக்கும் பன்மொழி செல்லப்பிராணி கதை உருவாக்கியை உருவாக்குதல்
- Foundry Local Spring Boot டெமோ மூலம் உள்ளூர் AI மாதிரி ஒருங்கிணைப்பை செயல்படுத்துதல்
- கணக்கிடும் எடுத்துக்காட்டுடன் மாடல் விதிமுறை சேவை (MCP) உருவாக்குதல்

## அறிமுகம்

இந்த அதிகாரம் ஜாவாவுடன் உருவாக்கும் உருவாக்கும் செயற்கை நுண்ணறிவு வடிவமைப்புகளை காட்டும் **மாதிரி திட்டங்களை** வெளிப்படுத்துகிறது. ஒவ்வொரு திட்டமும் முழுமையாக செயல்படக்கூடியதாகும் மற்றும் குறிப்பிட்ட AI தொழில்நுட்பங்கள், கட்டமைப்பு வடிவங்கள் மற்றும் சிறந்த நடைமுறைகளை நீங்கள் உங்கள் சொந்த பயன்பாடுகளுக்கு தழுவிக் கொள்ள முடியும்.

### Foundry Local Spring Boot டெமோ

**[Foundry Local Spring Boot டெமோ](foundrylocal/README.md)** உள்ளூர் AI மாதிரிகளுடன் **OpenAI Java SDK** மூலம் எப்படி ஒருங்கிணைக்கலாம் என்பதை காட்சி செய்கிறது. இது Foundry Local-ல் இயக்கப்படும் மாதிரிகளுடன் (பிரதமமாக **Phi-4-mini**) இணைக்கிறது, தானாக மாதிரி கண்டறிதல் வாயிலாக, மேக சேவைகளை சாராதே AI பயன்பாடுகளை இயக்க அனுமதிக்கிறது.

### நாய் கதை உருவாக்கி

**[நாய் கதை உருவாக்கி](petstory/README.md)** ஒரு ஈர்க்கக்கூடிய Spring Boot இணைய பயன்பாடாகும், இது பன்மொழி AI செயலாக்கத்தை காட்டி படைப்பாற்றல் சிறந்த செல்லப்பிராணி கதைகளை உருவாக்குகிறது. இது கிளையன்ட்-சைடு மற்றும் சர்வர்-சைடு AI திறன்களை இணைக்கும், கிளையன்ட்-பக்கம் AI தொடர்புகளுக்கு transformer.js மற்றும் சர்வர்-பக்கம் செயலாக்கத்திற்கு OpenAI SDK ஐ பயன்படுத்துகிறது.

### MCP கணக்கிடும் சேவை (ஆரம்பக்காரர் நட்பு MCP டெமோ)

**[MCP கணக்கிடும் சேவை](calculator/README.md)** Spring AI பயன்பாட்டுடன் **மாடல் விதிமுறை (MCP)** எளிய காட்சி ஆகும். இது MCP கருத்துக்களுக்கு ஆரம்பகட்ட அறிமுகத்தை அளிக்கிறது, ஒரு அடிப்படையான MCP சேவையகத்தை உருவாக்
