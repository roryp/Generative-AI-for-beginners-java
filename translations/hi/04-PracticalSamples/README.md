## आप क्या सीखेंगे
इस खंड में हम तीन व्यावहारिक अनुप्रयोगों का डेमो करेंगे जो Java के साथ जनरेटिव AI विकास पैटर्न दिखाते हैं:
- क्लाइंट-साइड और सर्वर-साइड AI को मिलाकर एक मल्टी-मोडल पेट स्टोरी जनरेटर बनाना
- Foundry Local Spring Boot डेमो के साथ स्थानीय AI मॉडल इंटीग्रेशन को लागू करना
- कैलकुलेटर उदाहरण के साथ एक मॉडल कॉन्टेक्स्ट प्रोटोकॉल (MCP) सेवा विकसित करना

## परिचय

यह अध्याय **नमूना परियोजनाओं** को प्रदर्शित करता है जो Java के साथ जनरेटिव AI विकास पैटर्न को प्रदर्शित करती हैं। प्रत्येक परियोजना पूरी तरह कार्यशील है और विशिष्ट AI तकनीकों, वास्तुकला पैटर्न, और सर्वोत्तम प्रथाओं को दिखाती है जिन्हें आप अपनी स्वयं की अनुप्रयोगों के लिए अनुकूलित कर सकते हैं।

### Foundry Local Spring Boot डेमो

**[Foundry Local Spring Boot डेमो](foundrylocal/README.md)** यह दर्शाता है कि **OpenAI Java SDK** का उपयोग करते हुए स्थानीय AI मॉडलों के साथ कैसे इंटीग्रेट किया जाए। यह Foundry Local पर चल रहे मॉडलों (जैसे, **Phi-4-mini**) से कनेक्ट करने को प्रदर्शित करता है, स्वचालित मॉडल पहचान के साथ, जिससे आप क्लाउड सेवाओं पर निर्भर रहे बिना AI अनुप्रयोग चला सकते हैं।

### पेट स्टोरी जनरेटर

**[पेट स्टोरी जनरेटर](petstory/README.md)** एक आकर्षक Spring Boot वेब एप्लिकेशन है जो **मल्टी-मोडल AI प्रोसेसिंग** के माध्यम से रचनात्मक पालतू कहानियाँ उत्पन्न करता है। यह क्लाइंट-साइड और सर्वर-साइड AI क्षमताओं को जोड़ता है, ब्राउज़र-आधारित AI इंटरैक्शन के लिए transformer.js और सर्वर-साइड प्रोसेसिंग के लिए OpenAI SDK का उपयोग करता है।

### MCP कैलकुलेटर सेवा (शुरुआती-मैत्रीपूर्ण MCP डेमो)

**[MCP कैलकुलेटर सेवा](calculator/README.md)** मॉडल कॉन्टेक्स्ट प्रोटोकॉल (MCP) का सरल प्रदर्शन है जो Spring AI का उपयोग करता है। यह MCP अवधारणाओं के लिए शुरुआती-मैत्रीपूर्ण परिचय प्रदान करता है, दिखाता है कि एक बुनियादी MCP सर्वर कैसे बनाया जाए जो MCP क्लाइंट्स के साथ इंटरैक्ट करता है।

## अध्ययन प्रगति

ये परियोजनाएँ पिछले अध्यायों की अवधारणाओं पर आधारित हैं:

1. **सरल से शुरू करें**: स्थानीय मॉडल के साथ बुनियादी AI एकीकरण समझने के लिए Foundry Local Spring Boot डेमो से शुरुआत करें
2. **इंटरैक्टिव बनाएं**: मल्टी-मोडल AI और वेब-आधारित इंटरैक्शन के लिए पेट स्टोरी जनरेटर पर जाएँ
3. **MCP मूल बातें सीखें**: Model Context Protocol के मूल सिद्धांतों को समझने के लिए MCP कैलकुलेटर सेवा का प्रयास करें

## सारांश

अच्छा काम! आपने अब कुछ वास्तविक अनुप्रयोगों का अन्वेषण किया है:

- मल्टी-मोडल AI अनुभव जो ब्राउज़र और सर्वर दोनों पर काम करते हैं
- आधुनिक Java फ्रेमवर्क और SDKs का उपयोग करके स्थानीय AI मॉडल एकीकरण
- आपका पहला Model Context Protocol सेवा यह देखने के लिए कि उपकरण AI के साथ कैसे एकीकृत होते हैं
