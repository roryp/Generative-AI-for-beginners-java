# പ്രായോഗിക പ്രയോഗങ്ങളും പ്രോജക്റ്റുകളും

## നിങ്ങൾ എന്താണ് പഠിക്കുക
ഈ വിഭാഗത്തിൽ, ജാവ ഉപയോഗിച്ച് ജനറേറ്റീവ് AI വികസന പാറ്റേണുകൾ പ്രദർശിപ്പിക്കുന്ന മൂന്ന് പ്രായോഗിക പ്രയോഗങ്ങൾ നമുക്ക് ഡെമോ ചെയ്യാം:
- ക്ലയന്റ്-സൈഡ്, സർവർ-സൈഡ് AI സംയോജിപ്പിച്ച് ഒരു മൾട്ടി-മോഡൽ Pet Story Generator സൃഷ്ടിക്കുക
- Foundry Local Spring Boot ഡെമോ ഉപയോഗിച്ച് പ്രാദേശിക AI മോഡൽ സംയോജനം നടപ്പിലാക്കുക
- Calculator ഉദാഹരണത്തോടെ ഒരു Model Context Protocol (MCP) സേവനം വികസിപ്പിക്കുക

## പരിചയം

ഈ അധ്യായത്തിൽ **സാമ്പിൾ പ്രോജക്റ്റുകൾ** പ്രദർശിപ്പിക്കുന്നു, ജാവ ഉപയോഗിച്ച് ജനറേറ്റീവ് AI വികസന പാറ്റേണുകൾ കാണിക്കുന്നു. ഓരോ പ്രോജക്റ്റും പൂർണ്ണമായും പ്രവർത്തനക്ഷമമാണ്, പ്രത്യേക AI സാങ്കേതികവിദ്യകൾ, ആർക്കിടെക്ചറൽ പാറ്റേണുകൾ, മികച്ച പ്രാക്ടീസുകൾ എന്നിവ പ്രദർശിപ്പിക്കുന്നു, നിങ്ങൾക്ക് നിങ്ങളുടെ സ്വന്തം പ്രയോഗങ്ങൾക്ക് ഇത് അനുകരിക്കാം.

### Foundry Local Spring Boot ഡെമോ

**[Foundry Local Spring Boot ഡെമോ](foundrylocal/README.md)** **OpenAI Java SDK** ഉപയോഗിച്ച് പ്രാദേശിക AI മോഡലുകളുമായി സംയോജിപ്പിക്കുന്നതെങ്ങനെ എന്ന് കാണിക്കുന്നു. ഇത് Foundry Local-ൽ പ്രവർത്തിക്കുന്ന **Phi-3.5-mini** മോഡലുമായി കണക്റ്റ് ചെയ്യുന്നത് പ്രദർശിപ്പിക്കുന്നു, ക്ലൗഡ് സേവനങ്ങൾ ആശ്രയിക്കാതെ AI പ്രയോഗങ്ങൾ പ്രവർത്തിപ്പിക്കാൻ നിങ്ങളെ അനുവദിക്കുന്നു.

### Pet Story Generator

**[Pet Story Generator](petstory/README.md)** ഒരു ആകർഷകമായ Spring Boot വെബ് പ്രയോഗമാണ്, **മൾട്ടി-മോഡൽ AI പ്രോസസ്സിംഗ്** പ്രദർശിപ്പിച്ച് സൃഷ്ടിപരമായ Pet Stories സൃഷ്ടിക്കുന്നു. ഇത് ബ്രൗസർ-അടിസ്ഥാനത്തിലുള്ള AI ഇടപെടലുകൾക്കായി transformer.js ഉപയോഗിക്കുകയും, സർവർ-സൈഡ് പ്രോസസ്സിംഗിനായി OpenAI SDK ഉപയോഗിക്കുകയും ചെയ്യുന്നു.

### MCP Calculator Service (ആരംഭക സൗഹൃദ MCP ഡെമോ)

**[MCP Calculator Service](calculator/README.md)** **Model Context Protocol (MCP)** ഉപയോഗിച്ച് ഒരു ലളിതമായ ഡെമോ ആണ്. ഇത് MCP ആശയങ്ങൾക്ക് ഒരു ആരംഭക സൗഹൃദ പരിചയം നൽകുന്നു, MCP ക്ലയന്റുകളുമായി ഇടപെടുന്ന ഒരു അടിസ്ഥാന MCP സെർവർ എങ്ങനെ സൃഷ്ടിക്കാമെന്ന് കാണിക്കുന്നു.

## പഠന പുരോഗതി

ഈ പ്രോജക്റ്റുകൾ മുൻ അധ്യായങ്ങളിൽ നിന്നുള്ള ആശയങ്ങൾ അടിസ്ഥാനമാക്കി രൂപകൽപ്പന ചെയ്തിരിക്കുന്നു:

1. **ലളിതമായി ആരംഭിക്കുക**: പ്രാദേശിക മോഡലുകളുമായി അടിസ്ഥാന AI സംയോജനം മനസിലാക്കാൻ Foundry Local Spring Boot ഡെമോ ഉപയോഗിച്ച് ആരംഭിക്കുക
2. **ഇടപെടലുകൾ ചേർക്കുക**: മൾട്ടി-മോഡൽ AIയും വെബ്-അടിസ്ഥാനത്തിലുള്ള ഇടപെടലുകളും മനസിലാക്കാൻ Pet Story Generator-ലേക്ക് പുരോഗമിക്കുക
3. **MCP അടിസ്ഥാനങ്ങൾ പഠിക്കുക**: Model Context Protocol അടിസ്ഥാനങ്ങൾ മനസിലാക്കാൻ MCP Calculator Service പരീക്ഷിക്കുക

## സംഗ്രഹം

നല്ല പ്രവർത്തി! നിങ്ങൾ ഇപ്പോൾ ചില യഥാർത്ഥ പ്രയോഗങ്ങൾ പരിശോധിച്ചു:

- ബ്രൗസറിലും സർവറിലും പ്രവർത്തിക്കുന്ന മൾട്ടി-മോഡൽ AI അനുഭവങ്ങൾ
- ആധുനിക ജാവ ഫ്രെയിംവർക്കുകളും SDKകളും ഉപയോഗിച്ച് പ്രാദേശിക AI മോഡൽ സംയോജനം
- ഉപകരണങ്ങൾ AI-യുമായി എങ്ങനെ സംയോജിപ്പിക്കാമെന്ന് കാണാൻ നിങ്ങളുടെ ആദ്യ Model Context Protocol സേവനം
