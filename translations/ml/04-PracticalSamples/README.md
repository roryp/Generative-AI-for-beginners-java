## നിങ്ങൾ പഠിക്കേണ്ടത്
ഈ ഭാഗത്തിൽ നമ്മൾ ജാവയുമായി ജനറേറ്റീവ് AI വികസന മാതൃകകൾ കാണിക്കുന്ന മൂന്ന് പ്രായോഗിക ആപ്ലിക്കേഷനുകൾ ഡെമോ ചെയ്യും:
- ക്ലയന്റ്-സൈഡ്‌യും സെർവർ-സൈഡ് AI യും സംയുക്തമാക്കി മൾട്ടി-മോഡൽ പേറ്റ് സ്റ്റോറി ജനറേറ്റർ സൃഷ്ടിക്കുക
- Foundry Local Spring Boot ഡെമോ ഉപയോഗിച്ച് ലോക്കൽ AI മോഡൽ ഇന്റഗ്രേഷൻ നടപ്പിലാക്കുക
- കാൽക്കുലേറ്റർ ഉദാഹരണവുമായി ഒരു മോഡൽ കോൺടെക്സ്‌റ പ്രോട്ടോകോൾ (MCP) സേർവീസ് വികസിപ്പിക്കുക

## പരിചയം

ഈ അധ്യായം **സാമ്പിൾ പ്രോജക്ടുകൾ** കാണിക്കുന്നു, ജാവയുമായി ജനറേറ്റീവ് AI ഡെവലപ്പ്മെന്റ് മാതൃകകൾ പ്രദർശിപ്പിക്കുന്നതു്. ഓരോ പ്രോജക്ടും പൂർണമായും പ്രവർത്തനക്ഷമമാണ്, AI സാങ്കേതികവിദ്യകൾ, ഘടനാ മാതൃകകൾ, ഉത്തമ പ്രവർത്തനരീതികൾ എന്നിവ നിങ്ങൾക്ക് നിങ്ങളുടെ ആപ്ലിക്കേഷനുകൾക്കായി അനുയോജ്യമായി മാറ്റാൻ കഴിയുന്നതായി കാണിക്കുന്നു.

### Foundry Local Spring Boot ഡെമോ

**[Foundry Local Spring Boot ഡെമോ](foundrylocal/README.md)** **OpenAI ജാവ SDK** ഉപയോഗിച്ച് ലോക്കൽ AI മോഡലുകളുമായി എങ്ങനെ ഇന്റഗ്രേറ്റ് ചെയ്യാമെന്ന് കാണിക്കുന്നു. Foundry Local-ൽ (ഉദാഹരണത്തിന്, **Phi-4-mini**) ഓടുന്ന മോഡലുകളുമായി കണക്റ്റ് ചെയ്യുന്നതും സ്വയം മോഡൽ കണ്ടെത്തൽ ലഭ്യമാക്കുന്നതും, ക്ലൗഡ് സർവീസുകളിൽ ആശ്രയിക്കാതെ AI ആപ്ലിക്കേഷനുകൾ പ്രവർത്തിപ്പിക്കാൻ സഹായിക്കുന്നതുമാണ്.

### പേറ്റ് സ്റ്റോറി ജനറേറ്റർ

**[പേറ്റ് സ്റ്റോറി ജനറേറ്റർ](petstory/README.md)** ഒരു ആകർഷകമായ സ്പ്രിംഗ് ബൂട്ടി വെബ് ആപ്ലിക്കേഷനാണ്, ഇൻവല്വ് ചെയ്യുന്ന **മൾട്ടി-മോഡൽ AI പ്രോസസ്സിംഗ്** ഉപയോഗിച്ച് സൃഷ്ടിപരമായ മൃഗ കഥകൾ ജനറേറ്റ് ചെയ്യാൻ. ബ്രൗസർ അടിസ്ഥാന AI ഇടപെടലുകൾക്കായുള്ള transformer.js ഉം സെർവർ-സൈഡ് പ്രോസസ്സിംഗിനായുള്ള OpenAI SDK ഉം ചേർത്തതാണ്.

### MCP കാൽക്കുലേറ്റർ സർവീസ് (ആരാധ്യമായ MCP ഡെമോ)

**[MCP കാൽക്കുലേറ്റർ സർവീസ്](calculator/README.md)** Spring AI ഉപയോഗിച്ച് **Model Context Protocol (MCP)** വിന്റെ ലളിതമായ ഡെമോയാണ്. MCP ആശയങ്ങൾക്ക് തുടക്കക്കാര-friendly പരിചയം നൽകുന്നു, MCP ക്ലയന്റുകളുമായും MCP സെർവറുമായി എങ്ങനെയാണ് അടിസ്ഥാന MCP സേർവർ സൃഷ്ടിക്കാമെന്ന് കാണിക്കുന്നു.

## പഠന പ്രവൃത്തി

ഈ പ്രോജക്ടുകൾ മുമ്പത്തെ അധ്യായങ്ങളിൽ നിന്നുള്ള ആശയങ്ങളെ അടിസ്ഥാനമാക്കി നിർമ്മിച്ചവയാണ്:

1. **ലളിതത്തിൽ തുടങ്ങുക**: ലോക്കൽ മോഡലുകളൊടുൾപ്പെടുന്ന മുമ്പിട്ട AI ഇന്റഗ്രേഷൻ മനസ്സിലാക്കാൻ Foundry Local Spring Boot ഡെമോ തുടക്കമാക്കുക
2. **ഇടപെടൽ കൂട്ടുക**: മൾട്ടി-മോഡൽ AIയുടെയും വെബ് അടിസ്ഥാന ഇടപെടലിന്റെയും വേണ്ടി പേറ്റ് സ്റ്റോറി ജനറേറ്ററിൽ പുരോഗമിക്കുക
3. **MCP അടിസ്ഥാനങ്ങൾ പഠിക്കുക**: മോഡൽ കോൺടെക്സ്‌റ് പ്രോട്ടോകോൾ അടിസ്ഥാനങ്ങൾ ചെയ്യാനായി MCP കാൽക്കുലേറ്റർ സർവീസ് പരീക്ഷിക്കുക

## സംക്ഷേപം

നല്ല പ്രവൃത്തി! നിങ്ങൾ ഇപ്പോൾ ചില യാഥാർത്ഥ്യത്തിലുള്ള ആപ്ലിക്കേഷനുകൾ പരിശോധിച്ചു:

- ബ്രൗസറിലും സെർവറിലും ഒന്നിച്ച് പ്രവർത്തിക്കുന്ന മൾട്ടി-മോഡൽ AI അനുഭവങ്ങൾ
- ആധുനിക ജാവ ഫ്രെയിമ്വർക്കുകളും SDK കളും ഉപയോഗിച്ച് ലോക്കൽ AI മോഡൽ ഇന്റഗ്രേഷൻ
- AI യുമായി ടൂളുകൾ എങ്ങനെ ഇന്റഗ്രേറ്റ് ചെയ്യുമെന്ന് കാണാൻ നിങ്ങളുടെ ആദ്യ Model Context Protocol സർവീസ്
