# പിന്തുണ

## പ്രശ്നങ്ങൾ റിപ്പോർട്ട് ചെയ്യാനും സഹായം നേടാനും എങ്ങനെ  

ഈ പ്രോജക്റ്റ് ബഗുകളും ഫീച്ചർ അഭ്യർത്ഥനകളും ട്രാക്ക് ചെയ്യാൻ GitHub Issues ഉപയോഗിക്കുന്നു. ഡ്യൂപ്ലിക്കേറ്റുകൾ ഒഴിവാക്കാൻ പുതിയ പ്രശ്നങ്ങൾ റിപ്പോർട്ട് ചെയ്യുന്നതിന് മുമ്പ് നിലവിലുള്ള പ്രശ്നങ്ങൾ തിരയുക. പുതിയ പ്രശ്നങ്ങൾക്കായി, നിങ്ങളുടെ ബഗ് അല്ലെങ്കിൽ ഫീച്ചർ അഭ്യർത്ഥന ഒരു പുതിയ Issue ആയി റിപ്പോർട്ട് ചെയ്യുക.

## മൈക്രോസോഫ്റ്റ് പിന്തുണാ നയം  

Generative AI for Beginners Java പ്രോജക്റ്റിനുള്ള പിന്തുണ മുകളിൽ ലിസ്റ്റുചെയ്തിരിക്കുന്ന വിഭവങ്ങളിൽ മാത്രം പരിമിതമാണ്.
