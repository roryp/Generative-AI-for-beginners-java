- Развој услуге Model Context Protocol (MCP) уз пример калкулатора

## Увод

Ово поглавље приказује **примерке пројеката** који демонстрирају обрасце развоја генеративне вештачке интелигенције помоћу Јаве. Сваки пројекат је потпуно функционалан и показује одређене AI технологије, архитектонске обрасце и најбоље праксе које можете применити у својим апликацијама.

### Foundry Local Spring Boot демо

**[Foundry Local Spring Boot демо](foundrylocal/README.md)** приказује како интегрисати локалне AI моделе користећи **OpenAI Java SDK**. Демонстрира повезивање са моделима који се извршавају на Foundry Local (нпр. **Phi-4-mini**) уз аутоматско откривање модела, омогућавајући вам коришћење AI апликација без ослањања на cloud услуге.

### Генератор прича о кућним љубимцима

**[Генератор прича о кућним љубимцима](petstory/README.md)** је занимљива веб апликација заснована на Spring Boot-у која демонстрира **мултимодалну AI обраду** за креирање креативних прича о кућним љубимцима. Комбинује могућности AI-а на клијентској и серверској страни користећи transformer.js за интеракције у прегледачу и OpenAI SDK за обраду на серверу.

### MCP Калкулатор услуга (Демо пријатељски почетницима)

**[MCP Калкулатор услуга](calculator/README.md)** је једноставан демонстрациони пројекат **Model Context Protocol (MCP)** користећи Spring AI. Пружа почетнички увод у MCP концепте, показујући како направити основни MCP сервер који комуницира са MCP клијентима.

## Прогрес учења

Ови пројекти су дизајнирани да надограде концепте из претходних поглавља:

1. **Почните једноставно**: Започните са Foundry Local Spring Boot демом да бисте разумели основну интеграцију са локалним моделима
2. **Додајте интерактивност**: Пређите на Генератор прича о кућним љубимцима за мултимодалну AI и веб интеракције
3. **Научите основе MCP-а**: Испробајте MCP Калкулатор услугу да разумете основе Model Context Protocol

## Резиме

Одличан посао! Сада сте истражили неке стварне апликације:

- Мултимодални AI доживљаји који раде и у прегледачу и на серверу
- Интеграција локалних AI модела користећи модерне Java фрејмворке и SDK-ове
- Вашу прву Model Context Protocol услугу да видите како алати интегришу AI
